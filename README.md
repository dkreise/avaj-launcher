# Aircraft Simulation Program (42 Java Project)

### How to Compile

```bash
find . -name "*.java" > sources.txt
javac @sources.txt
```

### How to Run

```bash
java ro.academyplus.avaj.simulator.Simulator scenarios/scenario.txt
```

### How to Clean

```bash
find . -name "*.class" -delete
rm -f sources.txt simulation.txt
```
